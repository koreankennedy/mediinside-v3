# 변경 이력 (Changelog)

모든 주요 변경사항을 기록합니다.

## [3.2.0] - 2026-01-04

### 버그 수정
- **거절횟수 글로벌 상태**: `useSyncExternalStore` 적용으로 페이지 이동 시에도 상태 유지
- **커스텀 이벤트 동기화**: `rejectCountChange` 이벤트로 같은 탭 내 실시간 동기화

### 개선
- **병원 프로필 완성하기 CTA**: 바이럴루프 팝업에서 중간 모달 제거, 프로필 완성도 모달로 직접 이동
- **재직경험 공유 서브탭**: 직원 리뷰 / 원장 재직경험 분리
- **리뷰 비밀번호 보호**: 공용 채용 계정 환경에서 리뷰 보호

### 기술적 변경
- `useGlobalRejectCount` 훅: `useState` → `useSyncExternalStore`로 리팩토링
- `handleResetLimit` 함수: `setShowProfileCTA(true)` → `router.push('/employer/profile?showCompletion=true')`

## [3.1.0] - 2026-01-02

### 추가
- **거절 10회 완료 모달**: AI 모의면접 보러가기 / 매칭 더 받기 선택
- **빈 화면 상태 UI**: 24시간 실시간 카운트다운 + 큰 CTA 버튼
- **재직경험 공유 CTA**: 리셋 모달에 4번째 옵션 추가
- **세션 평가 카운트**: 초기값 무관 실제 액션 기준 10회 추적

### 변경
- **거절 횟수 표시**: 버튼 내 작은 표시 → 버튼 하단 별도 영역
- **탭 순서**: 전체 → 진행 중 → 원장님 오퍼 → 인터뷰 요청
- **정렬 우선순위**: 진행 중 > 원장님 오퍼 > 매칭점수
- **모의면접 팝업**: 7회 → 10회 완료 시 표시

### UI/UX 개선
- **진행 중 탭 강조**: 불꽃 이모지, 파란색 테두리, animate-pulse
- **진행 중 카드 강조**: 파란색 테두리 + ring 효과 + 그림자
- **거절 횟수 경고**: 3회 이하일 때만 상단 경고 표시

## [3.0.0] - 2026-01-01

### 추가
- **업무환경 4단계**: 시설 → 근무 → 인력 → 장비
- **장비 선택 시스템**: 13개 진료과별 대표 장비 + 직접 입력
- **재직경험 공유 탭**: 프로필에 병원장/실장/동료 리뷰 작성
- **동료 리뷰 분리**: 전체평가 vs 개인별평가

### 변경
- **칸반 UI**: 색상 단순화, 폰트 계층 정리, 스크롤 버튼 추가
- **실시간 채용현황**: 연봉 정보 제외한 구체적 알림
- **홈 화면 구성**: 커리어 인사이트 그룹화
- **경력검증 수정 CTA**: 프로필 술기탭으로 연결

### UI/UX 개선
- **프로필 공개 범위**: 과거/재직 병원 숨김 옵션
- **리뷰 마스킹 요청**: 동료 리뷰 수정 요청 기능
- **조건 간단 수정**: 정밀진단 없이 희망조건 바로 수정

## [2.0.0] - 2025-12-xx

### 추가
- **AI 인터뷰 워크플로우**: 이력서 → 경력검증 → 모의면접 → AI면접 → 컨시어지
- **매칭 센터**: 제안/관심/거절 기능
- **AI 진단 (Fit Test)**: 성향 분석 기반 직장 추천
- **구인처 브랜딩**: 병원 브랜드 진단
- **후보자 관리**: 칸반 파이프라인

## 버전 네이밍 규칙

- **Major (X.0.0)**: 대규모 기능 추가 또는 아키텍처 변경
- **Minor (x.X.0)**: 새로운 기능 추가
- **Patch (x.x.X)**: 버그 수정 및 작은 개선
